pelaverse

code

“pelaverse” is a gitlab group that gather a bunch of packages created by pelagis. Its name is a reference to the tidyverse, “a collection of packages” helping to manipulate, visualize data. In the same spirit as tidyverse, pelaverse aims at gathering R packages used/created by pelagis. Those packages are mainly used for routine analysis of aerial campaign data.
